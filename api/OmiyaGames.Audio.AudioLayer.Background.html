    Declaration
    public IEnumerator PlayNextCoroutine(AssetRef<BackgroundAudio> audioClip, FadeInArgs fadeInArgs = null, FadeOutArgs fadeOutArgs = null, Action<AudioLayer.PlayerArgs> onFinish = null)
    Parameters
    Type Name Description
    AssetRef<BackgroundAudio> audioClip

    The clip to play
    FadeInArgs fadeInArgs

    Details on how to fade-in, e.g. how long it should last.
    FadeOutArgs fadeOutArgs

    Details on how to fade-out, e.g. how long it should last.
    Action<AudioLayer.PlayerArgs> onFinish

    Triggers as soon as this coroutine finishes, with a reference to the player that will be faded in. The parameter will be null if audioClip isn't played (e.g. when it's already playing in the background.)
    Returns
    Type Description
    IEnumerator

    A coroutine for loading audioClip.
    Remarks

    This coroutine needs to be yielded on regardless of whether audioClip points to an addressable or not.

    If audioClip's CurrentState is Null.
